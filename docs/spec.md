# Obsidian NWT Linker 仕様書

## 1. 概要

### 1.0 プラグイン名など
プラグイン名: nwt-linker
プラグインID: obsidian-nwt-linker

### 1.1 プラグインの目的

Obsidian上で入力された聖句参照を検出し、対応する jw.org の聖書リンクへ自動変換する。

例：

```txt
テトス1:14
```

を、一定時間後に自動で以下へ変換する。

```md
[テトス1:14](https://www.jw.org/finder?srcid=jwlshare&wtlocale=J&prefer=lang&bible=56001014&pub=nwtsty)
```

---

## 2. 基本方針

### 2.1 自動変換方式

ユーザーがコマンドを実行する方式ではなく、入力中のフォーカス行を監視し、自動変換する。

- フォーカス中の行を対象にする
- 入力後、1000ms のデバウンスを行う
- 1000ms 以上入力が止まった時点で解析する
- 聖句参照として完全な形になっている場合のみ変換する
- 不完全な入力中の文字列は変換しない

例：

```txt
テトス1:
```

これは変換しない。

```txt
テトス1:14
```

これは変換対象。

---

## 3. 対応する入力形式

### 3.1 基本文法

```txt
{書名}[半角/全角スペース任意]{章番号}:{節番号}
```

対応例：

```txt
テトス1:14
テトス 1:14
テトス　1:14
```

### 3.2 章節区切り

章と節の区切りはコロンのみ対応する。

対応：

```txt
テトス1:14
テトス１：１４
```

非対応：

```txt
テトス1章14節
テトス 1章14節
```

---

## 4. 正規化ルール

解析時には以下の正規化を行う。

ただし、**変換後の表示テキストは入力文字列をそのまま使う**。

### 4.1 正規化対象

| 入力 | 内部処理 |
|---|---|
| 全角数字 | 半角数字 |
| `：` | `:` |
| `，` | `,` |
| `；` | `;` |
| `−` | `-` |
| `–` | `-` |
| `—` | `-` |
| `ー` | `-` |

例：

```txt
テトス１：１４
```

内部的には：

```txt
テトス1:14
```

として解析する。

変換後の表示は：

```md
[テトス１：１４](...)
```

とし、元の入力表記を保持する。

## 5. 書名エイリアス

### 5.1 エイリアステーブル

書名と書名番号の対応はエイリアステーブルで管理する。

例：

```json
{
  "テトス": 56,
  "テト": 56,
  "啓示": 66,
  "啓": 66,
  "ヨハネ": 43,
  "ヨハ": 43,
  "ヨハネ第一": 62,
  "ヨハ一": 62
}
```

### 5.2 最長一致ルール

書名エイリアスは、文字数の長いものから順に最長一致で判定する。

これにより、次のような誤判定を防ぐ。

```txt
ヨハ一1:1
```

正しい解釈：

```txt
ヨハ一 + 1:1
```

誤った解釈：

```txt
ヨハ + 一1:1
```

---

## 6. bible ID生成ルール

リンクURLに含める `bible` パラメータは以下の形式で生成する。

```txt
{書名番号2桁}{章番号3桁}{節番号3桁}
```

### 6.1 パディング

- 書名番号：2桁
- 章番号：3桁
- 節番号：3桁

例：

```txt
テトス1:14
```

- テトス = 56
- 章 = 1 → 001
- 節 = 14 → 014

生成結果：

```txt
56001014
```

例：

```txt
詩編119:105
```

- 詩編 = 19
- 章 = 119
- 節 = 105

生成結果：

```txt
19119105
```

## 7. URL生成ルール

### 7.1 デフォルトURL

デフォルトでは以下の形式を使う。

```txt
https://www.jw.org/finder?srcid=jwlshare&wtlocale=J&prefer=lang&bible={{bible}}&pub=nwtsty
```

例：

```md
[テトス1:14](https://www.jw.org/finder?srcid=jwlshare&wtlocale=J&prefer=lang&bible=56001014&pub=nwtsty)
```

### 7.2 設定可能項目

以下は設定画面から変更可能にする。

```json
{
  "wtlocale": "J",
  "pub": "nwtsty",
  "urlTemplate": "https://www.jw.org/finder?srcid=jwlshare&wtlocale={{wtlocale}}&prefer=lang&bible={{bible}}&pub={{pub}}"
}
```

### 7.3 テンプレート変数

URLテンプレートでは以下の変数を使える。

| 変数 | 内容 |
|---|---|
| `{{bible}}` | bible ID |
| `{{wtlocale}}` | 言語ロケール |
| `{{pub}}` | 出版物ID |

### 8.1 入力

```txt
テトス1:14
```

### 8.2 出力

```md
[テトス1:14](https://www.jw.org/finder?srcid=jwlshare&wtlocale=J&prefer=lang&bible=56001014&pub=nwtsty)
```

## 9. 複数節の扱い

### 9.1 コンマ区切り

同じ章内の複数節指定に対応する。

対応例：

```txt
啓示21:3,4
啓示21:3, 4
啓示21:3,4,5
啓示21:3,5,7
```

### 9.2 空白の扱い

カンマ後の空白は許容する。

入力：

```txt
啓示21:3, 4
```

出力：

```md
[啓示21:3](url), [4](url)
```

可能な限り、入力時のカンマ後スペースを保持する。

### 9.3 表示形式

最初のリンクは書名・章・節を含める。

2つ目以降のリンクは、入力された節指定部分のみを表示する。

入力：

```txt
啓示21:3,5,7
```

出力：

```md
[啓示21:3](url),[5](url),[7](url)
```

入力：

```txt
啓示21:3, 5, 7
```

出力：

```md
[啓示21:3](url), [5](url), [7](url)
```

---

## 10. 範囲指定の扱い

### 10.1 基本範囲

範囲指定に対応する。

入力：

```txt
啓示21:3-5
```

リンク先は範囲の最初の節とする。

出力：

```md
[啓示21:3-5](https://www.jw.org/finder?srcid=jwlshare&wtlocale=J&prefer=lang&bible=66021003&pub=nwtsty)
```

### 10.2 複合指定

以下のような複合指定にも対応する。

```txt
啓示21:3-5,7,9-11
```

この場合、リンク先として使う節は以下。

| 表示 | リンク先の節 |
|---|---|
| `啓示21:3-5` | 3 |
| `7` | 7 |
| `9-11` | 9 |

出力：

```md
[啓示21:3-5](url3),[7](url7),[9-11](url9)
```

空白ありの場合：

```txt
啓示21:3-5, 7, 9-11
```

出力：

```md
[啓示21:3-5](url3), [7](url7), [9-11](url9)
```

---

## 11. 章またぎ表記

章をまたぐ指定には対応しない。

非対応例：

```txt
啓示21:3,22:1
啓示21:3-22:1
```

このような場合は、章またぎ部分を変換しない。

推奨挙動：

```txt
啓示21:3,22:1
```

は全体を変換しない。

理由：

- 誤変換を避けるため
- `22:1` を節指定として誤解釈しないため

---

## 12. セミコロン区切り

### 12.1 書名省略継承はしない

入力：

```txt
創 24:59; 35:8
```

この場合、`35:8` は `創` の続きとは解釈しない。

出力：

```md
[創 24:59](url); 35:8
```

### 12.2 セミコロン後に書名がある場合

書名が明示されている場合は変換する。

入力：

```txt
創 24:59; 出 3:2
```

出力：

```md
[創 24:59](url); [出 3:2](url)
```

---

## 13. 変換対象行の範囲

### 13.1 対象

フォーカス中の行を対象にする。

以下も変換対象とする。

```md
通常行 テトス1:14
- リスト テトス1:14
1. 番号付きリスト テトス1:14
> 引用 啓示21:3,4
| 聖句 | テトス1:14 |
```

### 13.2 非対象

コードブロック内は変換しない。

````md
```txt
テトス1:14
```
````

---

## 14. Markdown内での再変換防止

以下の中にある聖句参照は変換しない。

### 14.1 Markdownリンク内

```md
[テトス1:14](https://example.com)
```

### 14.2 URL部分

```md
[link](https://example.com/テトス1:14)
```

またはプレーンURL：

```txt
https://example.com/テトス1:14
```

### 14.3 インラインコード

```md
`テトス1:14`
```

### 14.4 コードブロック

````md
```txt
テトス1:14
```
````

### 14.5 Obsidian内部リンク

```md
[[テトス1:14]]
```

### 14.6 角括弧・丸括弧内

以下は変換しない。

```md
[テトス1:14]
[[テトス1:14]]
(テトス1:14)
```

## 15. 章節検証

### 15.1 検証方針

存在しない章節は変換しない。

書ごとの章数・節数データを持ち、変換前に検証する。

例：

```json
{
  "56": {
    "1": 16,
    "2": 15,
    "3": 15
  }
}
```

### 15.2 検証項目

以下を確認する。

- 書名番号が存在する
- 章番号が存在する
- 節番号が存在する
- 節番号がその章の最大節数以内である
- 範囲指定の場合、開始節と終了節が存在する
- 範囲指定の場合、開始節 <= 終了節
- 複数指定の場合、各節・各範囲がすべて有効である

### 15.3 無効な例

```txt
テトス9:99
啓示21:999
啓示21:5-3
啓示99:1
```

これらは変換しない。

---

## 16. エラー時の挙動

変換できない場合は何もしない。

- 通知しない
- エラー表示しない
- ステータスバーにも出さない
- 入力文字列をそのまま残す

例：

```txt
テトス9:99
```

無効な章節の場合、変換せずそのまま。

---

## 17. カーソル位置の扱い

自動変換後にカーソル位置を補正する。

### 17.1 基本ルール

変換対象がカーソルより前にある場合、変換前後の文字数差分ぶんカーソル位置を補正する。

例：

変換前：

```txt
今日読む テトス1:14｜
```

変換後：

```md
今日読む [テトス1:14](url)｜
```

カーソルはリンクの直後に移動する。

### 17.2 カーソルより後ろの変換

変換対象がカーソルより後ろにある場合、カーソル位置は原則変更しない。

---

## 18. Undoの扱い

自動変換は1回のUndoで元の文字列に戻せるようにする。

例：

変換後：

```md
[テトス1:14](url)
```

Undo 1回：

```txt
テトス1:14
```

### 18.1 実装上の要件

Obsidian / CodeMirror の1つのトランザクションとして置換を実行する。

複数箇所を同時に変換する場合も、同一行内の変換は可能な限り1つのUndo単位にまとめる。

---

## 19. 設定画面

設定画面のUIを作成する。

### 19.1 基本設定

| 項目 | 初期値 | 内容 |
|---|---:|---|
| プラグイン有効/無効 | 有効 | 自動変換を有効化する |
| デバウンス時間 | 1000ms | 入力停止後の解析待機時間 |
| wtlocale | `J` | jw.org の言語ロケール |
| pub | `nwtsty` | 出版物ID |
| URLテンプレート | デフォルトURL | リンク生成テンプレート |

### 19.2 エイリアス設定

書名エイリアスを設定画面で編集できるようにする。

最低限必要な機能：

- エイリアス追加
- エイリアス編集
- エイリアス削除
- 書名番号の指定
- JSONインポート
- JSONエクスポート
- デフォルトに戻す

章節データ設定UIとテスト入力欄は設けない。

## 20. データ構造案

### 20.1 設定データ

```json
{
  "enabled": true,
  "debounceMs": 1000,
  "wtlocale": "J",
  "pub": "nwtsty",
  "urlTemplate": "https://www.jw.org/finder?srcid=jwlshare&wtlocale={{wtlocale}}&prefer=lang&bible={{bible}}&pub={{pub}}",
  "aliases": {
    "テトス": 56,
    "テト": 56,
    "啓示": 66,
    "啓": 66
  }
}
```

### 20.2 解析結果データ

```ts
type ScriptureReference = {
  originalText: string;
  bookAlias: string;
  bookNumber: number;
  chapter: number;
  parts: ScripturePart[];
  startIndex: number;
  endIndex: number;
};

type ScripturePart = {
  originalText: string;
  startVerse: number;
  endVerse?: number;
};
```

例：

```txt
啓示21:3-5,7,9-11
```

解析結果イメージ：

```json
{
  "originalText": "啓示21:3-5,7,9-11",
  "bookAlias": "啓示",
  "bookNumber": 66,
  "chapter": 21,
  "parts": [
    {
      "originalText": "3-5",
      "startVerse": 3,
      "endVerse": 5
    },
    {
      "originalText": "7",
      "startVerse": 7
    },
    {
      "originalText": "9-11",
      "startVerse": 9,
      "endVerse": 11
    }
  ]
}
```

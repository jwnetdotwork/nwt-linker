## 4. Phase 2 後半: 再変換防止・カーソル補正・Undo制御

@docs/todo.md の以下を実装して。

- 2.7 再変換防止
- 2.9 カーソル位置補正
- 2.10 Undo制御
- 2.11 エラー時の無音化

要件:
- 既にMarkdownリンク化された聖句を再変換しない
- 変換対象がカーソルより前にある場合は文字数差分で補正する
- 変換対象がカーソルより後ろの場合はカーソル位置を維持する
- 同一行内の置換は可能な限り1つのCodeMirrorトランザクションにまとめる
- 変換できない場合は通知・エラー表示・ステータス表示を出さない

完了条件:
- 変換後にカーソルが不自然に移動しない
- Undo 1回で元の文字列に戻る
- 二重リンク化されない
- 無効入力でもUI通知が出ない
